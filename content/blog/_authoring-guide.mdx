## Adding a new post

1. Create a new file in `content/blog/`, e.g. `content/blog/why-yolo-deployments-fail.mdx`.
2. Add frontmatter at the top (between the `---` lines):

```yaml
---
title: "Why Most YOLO Deployments Fail in Production"
date: "2026-07-10"
summary: "And how to build ones that actually survive."
tags: ["Computer Vision", "Production AI"]
draft: false
---
```

3. Write the body in Markdown. Headings, **bold**, `inline code`, lists, links, and code blocks all render.

## What you can use

- `## Headings` and `### Subheadings`
- **Bold**, *italic*, and `inline code`
- Bullet and numbered lists
- > Blockquotes for callouts
- Fenced code blocks with triple backticks
- [Links](https://github.com/RH-NAYM)

```python
# code blocks are styled automatically
async def infer(image_url: str) -> dict:
    result = await model.predict(image_url)
    return result
```
